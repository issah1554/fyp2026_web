# Project Instructions

Follow `AGENTS.md`.

# Styling Rules

- Use Tailwind CSS utilities only for styling.
- Do not use inline styles, external CSS, CSS modules, styled-jsx, or component-level CSS files.
- Use only semantic Tailwind color tokens defined in `app/globals.css` (`main`, `primary`, `secondary`, `accent`, `success`, `warning`, `danger`, `info`, `pending`, `muted`).
- Do not use raw palette utilities, arbitrary color values, or hard-coded color values in component markup.
